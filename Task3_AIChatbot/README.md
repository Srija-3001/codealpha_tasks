# AI Chatbot

## Project Overview

AI Chatbot is a Java-based chatbot application developed as part of the CodeAlpha Java Programming Internship. The chatbot interacts with users through a console-based interface and provides responses to common questions using rule-based logic.

## Features

* Interactive conversation with users
* Responds to greetings
* Answers frequently asked questions
* Provides information about Java and CodeAlpha
* Simple and user-friendly console interface
* Exit option to end the conversation

## Technologies Used

* Java
* Scanner Class
* Conditional Statements
* Loops
* Object-Oriented Programming (OOP)

## Supported Queries

The chatbot can respond to questions such as:

* Hello / Hi
* What is your name?
* How are you?
* Tell me about Java
* Tell me about CodeAlpha
* What is the time?
* Bye

## How to Run

1. Compile the program:

```bash
javac AIChatbot.java
```

2. Run the program:

```bash
java AIChatbot
```

## Sample Output

The chatbot displays:

* Greeting messages
* Answers to predefined questions
* Information-based responses
* Exit message when the user types "bye"
