# Student Grade Tracker

## Project Overview

Student Grade Tracker is a Java-based application developed as part of the CodeAlpha Java Programming Internship. The program helps manage student grades, calculate performance statistics, and generate a summary report.

## Features

* Input student names and marks
* Store data using ArrayList
* Calculate average marks
* Find highest and lowest scores
* Assign grades based on marks
* Display a detailed student report

## Technologies Used

* Java
* ArrayList
* Scanner Class
* Object-Oriented Programming (OOP)

## Grade Criteria

| Marks Range | Grade |
| ----------- | ----- |
| 90 - 100    | A     |
| 80 - 89     | B     |
| 70 - 79     | C     |
| 60 - 69     | D     |
| Below 60    | Fail  |

## How to Run

1. Compile the program:

```bash
javac StudentGradeTracker.java
```

2. Run the program:

```bash
java StudentGradeTracker
```

## Sample Output

The program displays:

* Student Names
* Marks Obtained
* Grade Assigned
* Average Marks
* Highest Score
* Lowest Score
