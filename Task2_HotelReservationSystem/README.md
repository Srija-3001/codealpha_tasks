# Hotel Reservation System

## Project Overview

Hotel Reservation System is a Java-based application developed as part of the CodeAlpha Java Programming Internship. The system allows users to view available rooms, book rooms, cancel reservations, and manage hotel bookings through a simple console-based interface.

## Features

* View available rooms
* Book hotel rooms
* Cancel reservations
* View booking details
* Room categorization:

  * Standard
  * Deluxe
  * Suite
* Menu-driven console interface
* Object-Oriented Programming (OOP) implementation

## Technologies Used

* Java
* ArrayList
* Scanner Class
* Object-Oriented Programming (OOP)

## Room Categories

| Room Number | Room Type |
| ----------- | --------- |
| 101         | Standard  |
| 102         | Standard  |
| 201         | Deluxe    |
| 202         | Deluxe    |
| 301         | Suite     |

## How to Run

1. Compile the program:

```bash
javac HotelReservationSystem.java
```

2. Run the program:

```bash
java HotelReservationSystem
```

## Menu Options

1. View Rooms
2. Book Room
3. Cancel Reservation
4. View Bookings
5. Exit

## Sample Output

The program displays:

* Available rooms and booking status
* Successful booking confirmations
* Reservation cancellation messages
* Customer booking details
